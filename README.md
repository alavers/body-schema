> JSON schema based validation for express routes


## Install

```sh
$ npm install --save body-schema
```


## Usage

```js
var bodySchema = require('body-schema');

bodySchema('Rainbow');
```

```sh
$ npm install --global body-schema
$ body-schema --help
```
